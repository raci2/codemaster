Perguntas sobre HTML

Alexandru Racicovschi

1 - O que é HTML?

HTML (HyperText Markup Language) é a linguagem usada para estruturar e apresentar conteúdo na web. Define a estrutura de uma página web através de elementos como cabeçalhos, parágrafos, links e imagens, que são interpretados pelos navegadores para mostrar o conteúdo corretamente.

2 - O que é um elemento HTML?

Um elemento HTML é uma unidade básica de um documento HTML que define a estrutura e o conteúdo de uma página web. Cada elemento é composto por uma tag de abertura, o conteúdo e uma tag de fechamento. Por exemplo, o elemento <p> define um parágrafo, com a tag de abertura <p>, o conteúdo do parágrafo e a tag de fechamento </p>. Alguns elementos, como <img>, não têm uma tag de fechamento.

3 - Que ferramentas são necessárias para construir um site HTML?

Para construir um site em HTML, precisamos das seguintes ferramentas básicas:

  • Computador
  • Editor de texto: VSCode, bloco de notas...
  • Navegador: Para visualizar o site, Google Chrome, firefox...

4 - Quais os elementos base necessários para a criação de uma página HTML?

Os elementos base necessários para a criação de uma página HTML são:
<!DOCTYPE html>
<html>
<head> </head>
<body> </body>
</html>

5 - O elemento br necessita ser fechado?

O elemento <br> não necessita de ser fechado, também como o elemento <img>,<a>, entre outros. Esses elementos são vazios e não possuem contéudo.

6 - Para que servem os atributos?

Os atributos em HTML servem para fornecer informações adicionais sobre os elementos. Eles são usados para definir características e comportamentos de um elemento, como o seu estilo, identificação, classe, ou o destino de um link. Os atributos são sempre especificados na tag de abertura de um elemento.
Exemplo de Atributos:

  1. lang
  2. charset
  3. href
  4. src
  5. alt
